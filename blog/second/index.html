Hugo Cheat Sheet

September 7, 2024

Here’s a list of useful hugo commands I regularly use.

# Build your hugo project
hugo

# Create a new content file
hugo new <path>
# For example: hugo new posts/hugo-cheat-sheet.md

# Run the hugo development server
hugo server

Remember to set draft = false for content to be displayed.
